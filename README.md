Cachegoose + Redis

Showdown

# Usage
## Run!
First of all you need redis and mongodb set up.
then...
```
npm install
node app/app
```
Visit http://localhost:7788/
## Build Front-end
Use Gulp , Uglify and ng-Annotate when Packing
```
cd web/src
npm install
gulp default
```
